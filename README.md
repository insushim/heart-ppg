# Heart — 카메라 PPG 심박·컨디션 측정 (Android)

스마트폰 후면 카메라 + 플래시에 손가락을 대고 측정하는 PPG(광혈류측정) 기반 웰니스 앱.

> ⚠️ **웰니스/참고용입니다. 의료기기가 아니며 질병의 진단·치료·예방 목적으로 사용할 수 없습니다.**
> SpO2(산소포화도)·혈압은 스마트폰 카메라로 신뢰성 있게 측정할 수 없어 **의도적으로 제외**했습니다.

## 측정 항목 (4계열 교차검증으로 신뢰성 확정된 것만)
- 심박수 (bpm)
- 심박변이도 HRV (RMSSD, SDNN, pNN50)
- 자율신경 컨디션 (신호등 — 절대 점수 미표시)
- 호흡수 (안정·정지 조건 충족 시)
- 관류지수 PI (신호 품질 지표)
- 불규칙 맥박 스크리닝 (진단 아님)

## 구조
- `core/` — 순수 Kotlin/JVM 신호처리(FFT·밴드패스·피크검출·HR/HRV/호흡/PI). 합성신호 단위테스트로 검증. 하드웨어 불필요.
- `app/` — Android (Jetpack Compose + CameraX). 카메라 red 채널 추출 → `core` 분석.

## 빌드
```bash
./gradlew :core:test          # 신호처리 알고리즘 검증
./gradlew :app:assembleDebug  # 디버그 APK (Android SDK 필요)
```
CI(GitHub Actions)가 푸시마다 테스트 + APK 빌드 + Release 발행.

## 정확도 주의
정확도는 기기·조명·피부 상태·움직임에 따라 달라집니다. 표시되는 수치는 실험실 조건 기준이 아니며 보장값이 아닙니다.
